Cloner un repository :
git clone "url du repository"

Faire un push :

git add .
git commit -m "nom du commit"
git push

Pour récupérer les dernieres mises à jour sur github sur vs code
git pull

Pour créer une nouvelle branche et basculer directement dessus :
git checkout -b "nom de la branche"
ou
git switch -c nom de la branche

Pour regarder dans quelle branche on est :
git branch

Pour aller sur une branche :
git checkout "nom de la branche"
ou
git switch nom de la branche

fin du tuto
